# 前端 UI 优化文档

## 概述

对网亿 API 管理插件的前端页面进行了全面的 UI/UX 优化，使其更好地适配子比主题的设计风格。

## 优化内容

### 1. API 列表页面优化

**文件**: `public/templates/api-list.php`

#### 优化点：
- 现代化卡片式设计，渐变色彩方案
- 左侧分类导航 + 统计卡片
- API 方法标签采用渐变色（GET/POST/PUT/DELETE）
- 悬停效果优化（卡片上浮 + 阴影增强）
- 响应式布局，移动端友好

**配色方案**：
- 主色调：`#667eea` → `#764ba2`（紫色渐变）
- 强调色：`#206bcc`（蓝色）
- 成功色：`#2e7d32`
- 警告色：`#f57f17`
- 错误色：`#c62828`

### 2. API 详情页优化

**文件**: `public/css/api-detail.css`（新增）

#### 新增组件：
1. **返回导航** - 快速返回列表页
2. **头部卡片** - 渐变背景，突出 API 名称和方法
3. **分区布局** - 左侧主内容 + 右侧测试区
4. **参数表格** - 清晰的表格样式，带类型和必填标签
5. **状态码卡片** - 彩色区分的状态码展示
6. **代码示例** - 标签页切换的代码块
7. **测试表单** - 现代化的表单输入和响应显示

#### 交互优化：
- 所有按钮增加悬停效果（上浮 + 阴影）
- 表单输入框聚焦时有蓝色光晕
- 代码块支持一键复制
- 响应区域显示耗时和状态码

### 3. 深色模式适配

两个页面都完全适配子比主题的深色模式（`body.dark-theme`）

**深色模式特点**：
- 背景色：`#1E2029`（深蓝灰）
- 卡片色：`#1E2029` → `#272935`
- 文字色：`#E5E7EB` / `#9CA3AF` / `#DFE1E5`
- 保持浅色模式的视觉层次
- HTTP 方法标签使用深色渐变

### 4. 响应式设计

#### 断点：
- `992px` - 平板和桌面切换
- `768px` - 平板和手机切换
- `576px` - 大屏手机和小屏手机切换

#### 响应式行为：
- 列表页：侧边栏在小屏时移到顶部
- 详情页：左右布局在平板变为上下布局
- 卡片网格自适应列数
- 状态码卡片自动换行

## 技术实现

### CSS 架构
```css
/* 1. 基础样式 - 组件结构 */
.api-section { ... }

/* 2. 布局样式 - Grid/Flex */
.api-detail-layout { display: grid; }

/* 3. 交互样式 - 悬停/聚焦 */
.submit-btn:hover { ... }

/* 4. 深色模式 - body.dark-theme 限定 */
body.dark-theme .api-section { ... }

/* 5. 响应式 - @media 查询 */
@media (max-width: 992px) { ... }
```

### 渐变方案
```css
/* 主渐变 - 用于重要按钮和头部 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 辅助渐变 - 用于背景装饰 */
background: linear-gradient(135deg, #f8f9fa 0%, #fff 100%);
```

### 图标使用
使用 Font Awesome 图标（需主题支持）：
- `<i class="fa fa-bolt"></i>` - API 图标
- `<i class="fa fa-play"></i>` - 测试按钮
- `<i class="fa fa-copy"></i>` - 复制按钮
- `<i class="fa fa-arrow-left"></i>` - 返回按钮
